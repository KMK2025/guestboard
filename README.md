# 간단한 게시판

HTML, CSS, JavaScript로 만든 간단한 게시판입니다.  
Node.js(Express)를 사용하여 서버 로컬(`posts.json`)에 게시글을 저장하므로, 서버를 다시 실행해도 게시글이 유지됩니다.

## 기능

- **게시글 작성**: 제목, 작성자, 내용, 비밀번호 입력 후 **글게시** 버튼으로 등록
- **게시글 목록**: 목록에서 제목, 내용, 작성자, 작성일 확인
- **수정 / 삭제**: 글 작성 시 설정한 **비밀번호**를 아는 사람만 해당 글을 수정/삭제 가능

## 준비사항

- Node.js (v14 이상 권장)

## 설치 및 실행 방법

1. 이 폴더에서 필요한 패키지 설치

   ```bash
   npm install
   ```

2. 서버 실행

   ```bash
   npm start
   ```

3. 브라우저에서 접속

   - 주소: `http://localhost:3000`

4. 사용 방법

   - 상단의 **새 글 작성** 폼에서
     - 제목, 작성자, 비밀번호, 내용을 입력 후 **글게시** 버튼 클릭
   - 하단 목록에서
     - **수정**: 비밀번호를 입력하면 제목/내용을 변경 가능
     - **삭제**: 비밀번호를 입력하면 해당 글 삭제

## 데이터 저장 위치

- 게시글은 이 폴더의 `posts.json` 파일에 JSON 형식으로 저장됩니다.
- 파일이 없으면 첫 실행 시 자동으로 생성됩니다.
